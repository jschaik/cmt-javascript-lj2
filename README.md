# Periode 1 — JavaScript Fundament

## Software Development MBO 4 · Leerjaar 2

Welkom bij de opdrachtenrepository voor periode 1.

## Hoe gebruik je deze repository?

1. Klik op 'Use this template' → 'Create a new repository'
2. Kies je eigen GitHub-account als eigenaar
3. Geef de repository een naam: periode-1-[jouw-naam]
4. Clone de repository naar je computer
5. Maak elke week een nieuwe branch: week-01/oefeningen

## Structuur per week

Elke week heeft drie mappen:

- oefening-1/   → ~45 minuten, één concept geïsoleerd
- oefening-2/   → ~45 minuten, één concept geïsoleerd
- eindopdracht/ → ~4 uur, alles combineren

Klaar met de eindopdracht? Stijl de app met CSS.

## Inleveren

1. Commit en push je werk naar GitHub
2. Kopieer de link naar je branch
3. Lever de link in via Canvas

## Branches

Gebruik altijd dit formaat:

```
week-01/oefeningen
week-02/oefeningen
week-03/oefeningen
...
```
