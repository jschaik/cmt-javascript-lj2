# Week 7 — Herhaling fetch & debuggen

## Opdracht

In elke debug-map zit een kapotte fetch-app. Zoek de fout, fix hem, en schrijf de fout als commentaar bovenaan `script.js`.

## Instructie

1. Open de app in de browser
2. Lees de foutmelding in de console
3. Zoek de fout in de code
4. Fix de fout
5. Schrijf bovenaan `script.js`: `// Fout was: ...`
