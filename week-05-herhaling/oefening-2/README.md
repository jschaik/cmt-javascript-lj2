# Oefening 2 Week 5 — Speler aanmaken via formulier

## Opdracht (~1 uur)

Een formulier met twee velden: naam en score.

1. Luister naar het submit-event op het formulier.
2. Valideer de invoer: naam mag niet leeg zijn, score moet een getal zijn.
3. Maak een nieuw speler-object aan met **spread** op het standaardobject en de ingevulde waarden.
4. Voeg het toe aan de array met `push` en herrender de lijst.

## Concepten

- object literals
- spread
- `addEventListener`
- validatie
- `array.push`
