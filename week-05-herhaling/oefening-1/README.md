# Oefening 1 Week 5 — Spelers filteren en sorteren

## Opdracht (~1 uur)

Gegeven een array van speler-objecten (naam, score, level).

1. Gebruik `filter()` om alleen spelers met een score **boven 100** te houden.
2. Gebruik `sort()` om ze van **hoog naar laag** op score te sorteren.
3. Toon het resultaat als een lijst in de DOM met `map()` en `innerHTML`.

## Concepten

- `filter()`
- `sort()`
- `map()`
- `innerHTML`
- objecten
